# 🎓 Student Portfolio & Registration System

A simple full-stack web application that allows students to **register, log in, and create a personal portfolio** to showcase their skills, education, and projects.

---

## 🚀 Features

* 🔐 User Registration & Login
* 📄 Personal Portfolio Dashboard
* 🧠 Display Skills, Education & Projects
* ✅ Form Validation (JavaScript)
* 🔄 Session Management (PHP)
* 💾 Data Storage using JSON

## ⚙️ Setup Instructions

### 1️⃣ Install XAMPP

Download and install **XAMPP**.

### 2️⃣ Move Project

Place your project folder inside:

```
C:\xampp\htdocs\
```

### 3️⃣ Start Server

Open XAMPP Control Panel:

* Start **Apache**

### 4️⃣ Run Project

Open browser:

```
http://localhost/student-portfolio/
```

---

## 🧪 How It Works

1. User registers using the registration form
2. Data is stored in `users.json`
3. User logs in with credentials
4. Session is created
5. Portfolio page fetches and displays user data
